# writeups
Aquí publicare los writeups de mi experiencia con maquinas de Hack The Box y Try Hack Me. En ellas vereis tanto el writeup como mi distribucion típica de carpetas cuando realizo maquinas (nmap/content/scripts/exploits) si hubiera archivos dentro.

# nmap:
Comandos de nmap, el de identificacion de puertos en formato grepeable y el especifico para lanzar los scripts nse.

# content: 
Cualquier contenido que vaya ligado a la máquina o descarguemos de ella.

# scripts: 
Scripts que crearemos para explotar vulnerabilidades o identificarlas.

# exploits: 
Aquí se guardaran los exploits/CVEs encontrados en la web para una vulnerabilidad concreta que encontremos.
